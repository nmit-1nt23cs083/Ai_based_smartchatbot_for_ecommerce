📋 THE 9 EVENTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. REGISTER          → User signs up           → TREND20 (20% OFF)
2. LOGIN_WELCOME    → User logs in            → Welcome back
3. CART_ADDED       → Item to cart            → TREND20 (20% OFF)
4. CART_REMINDER    → After 3 minutes         → SAVE30 (30% OFF)
5. ORDER_PLACED     → Checkout complete      → Confirmation + items
6. ORDER_PROCESSING → Admin marks processing → Status update
7. ORDER_SHIPPED    → Admin marks shipped    → Tracking number
8. ORDER_OUT_OF_DELIVERY → Admin marks for delivery → Today delivery
9. ORDER_DELIVERED  → Admin marks delivered  → Review request

Every notification logs like this:

[register] ✅ Email | ✅ WhatsApp | User: john@example.com | Phone: +919876543210

Breakdown:
  [register]                    = Event type
  ✅ Email                      = Email sent successfully
  ✅ WhatsApp                   = WhatsApp sent successfully
  User: john@example.com        = Customer email
  Phone: +919876543210          = WhatsApp target (formatted)

This helps with debugging! 🔍
