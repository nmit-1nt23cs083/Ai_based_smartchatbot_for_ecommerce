# 🚀 AI Recommendation System - Quick Start Guide

## What's New?
Your Trendify project now has an intelligent recommendation engine that:
- Tracks user behavior (searches, cart, wishlist)
- Generates AI-powered personalized messages
- Sends targeted Telegram notifications with product links
- Makes voice calls for order confirmations
- Automatically sends recommendations every 2-5 minutes

---

## 🎯 How It Works

### 1. **User Behavior Tracking** 📊
Every user action is logged:
```javascript
// In frontend (api.js)
Tracking.trackSearch("winter jackets");
Tracking.trackProductView(123, "Blue Jacket");
Tracking.trackAddToCart(123, "Blue Jacket");
Tracking.trackAddToWishlist(456, "Red Sweater");
```

### 2. **Scheduled Reminders**
- **Every 2 minutes**: Finds abandoned carts → sends reminder via Telegram
- **Every 3 minutes**: Detects searches without purchase → offers discount
- **Every 5 minutes**: Alerts users with wishlist items on sale

### 3. **Order Confirmation Flow**
When a user places an order:
1. ✅ Telegram confirmation with "Track Order" button
2. 📞 Voice call with order details
3. ⏰ 4-minute delay
4. 💡 Personalized product recommendations via Telegram

## 📱 Frontend Integration

### Add Tracking to Your Pages

**In shop.html - Track Searches:**
```html
<input type="search" id="search-box" placeholder="Search...">

<script>
  document.getElementById('search-box').addEventListener('change', (e) => {
    Tracking.trackSearch(e.target.value);
    // ... then do search
  });
</script>
```

**In product.html - Track Product Views:**
```html
<script>
  // On page load
  const productId = new URLSearchParams(window.location.search).get('id');
  const productName = document.querySelector('h1').textContent;
  Tracking.trackProductView(productId, productName);
</script>
```

**In cart - Track Add/Remove:**
```html
<button onclick="addToCart(productId, productName)">
  Add to Cart
</button>

<script>
  async function addToCart(id, name) {
    // ... your cart code
    Tracking.trackAddToCart(id, name);
  }
</script>
```

## 🎛️ Admin Panel Features

### Send Personalized Messages
```
Admin Dashboard → Users → Actions Menu → "Personalize User"
```
This reads the user's behavior and sends a custom-tailored Telegram message with their cart/wishlist items.

### Send Recommendations  
```
Admin Dashboard → Users → Actions Menu → "Generate Recommendations"
```
AI generates recommendations based on user's past purchases and sends via Telegram.

### Send Order Confirmation
```
Admin Dashboard → Orders → "Confirm Selection"
```
Sends order confirmation + makes voice call + schedules recommendations.

## 🤖 Environment Setup

Your `.env` file should have:

```env
# Telegram (for messages)
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id

# Groq AI (for message generation)
GROQ_API_KEY=your_groq_api_key

# Twilio (for voice calls)
TWILIO_ACCOUNT_SID=your_twilio_sid
TWILIO_AUTH_TOKEN=your_twilio_token
TWILIO_PHONE_NUMBER=+1234567890
USER_PHONE_NUMBER=+1234567890
```

## 📊 API Endpoints

### Track User Action
```bash
POST /api/traces
{
  "action": "search",           # or view, add_to_cart, etc
  "target_type": "product",
  "target_id": 123,
  "target_name": "Blue Jacket",
  "page_url": "/shop.html",
  "session_id": "abc123"
}
```

### Get User Traces (Admin)
```bash
GET /api/traces?limit=100
# Returns all tracked user actions
```

### Send Personalized Message
```bash
POST /api/admin/users/123/send-personalized-message
# Analyzes user's cart/wishlist and sends targeted Telegram
```

### Send Recommendations
```bash
POST /api/admin/users/123/send-recommendations
# Generates AI recommendations based on user's purchases
```

## 🔧 Customization

### Change Reminder Timings
Edit `reminderService.js`:
```javascript
setInterval(async () => {
  // Cart scan - currently 2 minutes
}, 2 * 60 * 1000);  // Change to 5 * 60 * 1000 for 5 minutes

setInterval(async () => {
  // Search scan - currently 3 minutes
}, 3 * 60 * 1000);

setInterval(async () => {
  // Wishlist scan - currently 5 minutes
}, 5 * 60 * 1000);
```

### Edit Message Templates
The messages are generated by Groq AI, but you can customize the prompts in:
- `generateTwoMessages()` - for cart/search/wishlist reminders
- `generateOrderMessages()` - for order confirmations
- `generateRecommendationMessage()` - for product recommendations

## ❓ Troubleshooting

**"Telegram messages not sending"**
- Check TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID in .env
- Make sure bot is added to the chat

**"Voice calls not working"**
- Verify TWILIO credentials in .env
- Check USER_PHONE_NUMBER format (+country_code_number)

**"No recommendations generated"**
- Check GROQ_API_KEY is valid
- Look for errors in server logs
- System has fallback messages if Groq fails

**"Interval training not running"**
- Check server logs for "🚀 [Interval Training] Started!"
- Restart the server: `node server.js`
